# Venchidex Electrical Website

A responsive React-based website for **Venchidex Electrical**, showcasing LED, solar, and street lighting solutions.  
Built with **React Router**, **Helmet for SEO**, and modular CSS for clean styling.

---

## 🚀 Features
- Responsive design (mobile, tablet, desktop)
- Sticky header with hamburger menu
- Floating WhatsApp button
- Back-to-top button
- SEO metadata per page (via Helmet)
- Product catalog with animations
- Testimonials + FAQ section
- Contact form with WhatsApp integration
